# 路上接龍

給 iPhone 乘客使用的語音成語接龍。遊戲、AI 與成語庫不依賴外部服務；語音辨識由 Safari／iOS 提供，離線時不保證可用。

## 遊戲模式

- 同字接龍：下一個成語必須以目標字開頭，成功得 1 分。
- 同音接龍：下一個成語可用目標字或同音字開頭，成功得 1 分。
- 多人積分：同字在字首 3 分；同音字在字首 2 分；同字或同音字在其他位置 1 分；都沒有則 0 分。只要是未使用過的內建成語，回合仍會繼續。
- 單人模式：玩家與本機 AI 輪流，可選輕鬆、標準或挑戰難度。
- 多人模式：2–6 位乘客在同一支手機輪流。
- 開局方式：可由系統從 2,600 多個適合接續的成語中隨機出題，或由玩家自由說第一個四字詞開局。

遊戲中可以說「再說一次」、「提示」或「結束遊戲」。多人模式說「放棄」後，必須再說「確定」才會以 0 分換下一位；說「取消」則繼續原回合。Safari 沒聽清楚時，也能使用畫面下方的文字輸入。

Safari 若把同音字寫錯，遊戲會用教育部音讀比對詞庫；唯一候選會自動修正，多個候選則以語音詢問「確定」或「不是」。

「同音」目前以國語聲母、韻母與聲調都相同判定。

## 成語資料

遊戲目前可辨識 37,548 條四字詞目：5,395 條成語，以及 32,153 條其他四字詞語。來源包含中華民國教育部《成語典》`dict_idioms_2020_20260324`、《國語辭典簡編本》`dict_concised_2014_20260325`與《重編國語辭典修訂本》`dict_revised_2015_20260325`，並保留原始詞目與漢語拼音。資料依 CC BY-ND 3.0 TW 使用，完整來源與使用說明位於 `licenses/`。

位置積分賽中，成語依原規則取得 3／2／1 分；只收錄於一般辭典的四字詞語最高得 1 分。

如需更新官方資料：

```powershell
python scripts/import_moe_idioms.py <教育部下載的 xlsx> moe-idioms.js
python scripts/import_moe_phrases.py <簡編本 xlsx> <重編本 xlsx> moe-phrases.js
```

## 使用方式

```powershell
npm run serve
```

瀏覽器開啟 `http://localhost:4173`。部署至 HTTPS 網站後，iPhone 才能正常請求麥克風權限並加入主畫面。第一次載入後，靜態遊戲資源會快取供離線使用。

## 安裝到 iPhone 主畫面

1. 以 Safari 開啟部署後的 HTTPS 網址。
2. 點 Safari 的分享按鈕。
3. 選擇「加入主畫面」，再按「新增」。

專案包含 `.github/workflows/deploy-pages.yml`。推送至 GitHub 的 `main` 分支後，GitHub Actions 會先執行測試，再部署 Pages。

連線狀態以遊戲主機上的 `online-check.txt` 實際確認；行動網路、Wi-Fi 與個人熱點都適用。語音辨識若回報網路錯誤，遊戲會保留語音播報並顯示文字輸入備援。
